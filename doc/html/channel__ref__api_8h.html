int channel_ref_accept ( channel_ref_t *  channel_ref,
const char *  ip,
int  port,
int  backlog 
)

将管道转换为监听管道 由这个监听管道接受的新连接将使用与监听管道相同的发送缓冲区最大数量限制和接受缓冲区长度限制, channel_ref_accept所接受的新连接将被负载均衡，实际运行在哪个loop_t内依赖于实际运行的情况
